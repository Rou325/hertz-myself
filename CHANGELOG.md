## v2.1.0 (2026-06-13)

### 🎯 新增功能

#### 交互式配置系统
- 选项模式：用户只需输入数字（1/2/3）
- 支持多种配置：默认配置、API 配置、跳过配置
- 自定义触发时间

#### 多种 API 支持
- Exa：AI 优化的搜索引擎（国内友好）
- Tavily：AI 优化的搜索引擎（国内友好）
- Spotify：专业音乐数据（国内不友好）
- WebSearch：默认方案（完全免费）

#### 多维度评分系统
- 评分维度：评分、心情、场景、时间
- 偏好学习：根据评分记录用户喜好
- 统计分析：详细的评分统计

## v2.0.0 (2026-06-13)

### 🎯 新增功能

#### 智能推荐系统
- 实时搜索：根据对话内容实时搜索歌曲
- 情绪分析：分析对话情绪和主题
- 随机性强：每次推荐都不同

#### 用户评分系统
- 评分记录：记录用户对歌曲的评分
- 偏好学习：根据评分记录用户喜好
- 个性化推荐：基于历史评分优化推荐

#### 智能过滤
- 过滤抖音热歌：自动过滤网络神曲、口水歌
- 追求品质：推荐歌词有深度、旋律有特色的歌曲
- 小众佳作：优先推荐独立音乐、冷门宝藏
