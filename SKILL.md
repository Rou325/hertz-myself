# 🎵 听见自己的频率 - 智能音乐推荐技能

根据当天的对话历史，智能分析内容，实时搜索推荐歌曲。

## 核心特点

### ✨ 智能推荐
- **实时搜索**：根据对话内容实时搜索歌曲，不使用预设数据库
- **情绪分析**：分析对话情绪和主题，推荐最合适的歌曲
- **随机性强**：每次推荐都不同，探索新音乐

### 🚫 拒绝热歌
- **过滤抖音热歌**：自动过滤网络神曲、口水歌
- **过滤网红歌**：避免网红歌曲、变装BGM、挑战BGM
- **追求品质**：推荐歌词有深度、旋律有特色的歌曲
- **小众佳作**：优先推荐独立音乐、冷门宝藏
- **全球范围**：不限语言，中文、英文、日文、韩文等均可

### 📊 用户反馈
- **评分系统**：用户听完后可评分（1-10分）
- **偏好学习**：根据评分记录用户喜好
- **个性化推荐**：基于历史评分优化推荐

### 🌤️ 天气集成
- **自动检测**：检测是否有天气 skill 可用
- **智能结合**：结合天气状况优化推荐
- **无感使用**：有天气 skill 就用，没有就跳过

### 🎭 人格集成
- **首次询问**：首次调用时询问用户是否使用人格文件
- **用户选择**：如果选择使用，由用户提供人格文件路径
- **套用风格**：读取人格文件，套用语言风格

## 工作流程

### 1. 对话分析
```
读取当天对话历史
↓
检测天气（如果有天气 skill）
↓
分析情绪（结合天气）
↓
识别主题（工作/生活/学习/娱乐/爱情）
```

### 2. 生成搜索建议
```
根据情绪和主题
↓
生成搜索关键词
↓
考虑用户偏好
↓
输出搜索指令
```

### 3. 实时搜索
```
AI 根据搜索指令
↓
在网络上搜索歌曲
↓
过滤抖音热歌
↓
推荐 1 首歌曲
```

### 4. 用户评分
```
用户听完歌曲
↓
给出评分（1-10分）
↓
记录到数据库
↓
更新用户偏好
```

## 使用方法

### 方式1：首次配置（推荐）
首次调用 hertz-myself 时，系统会询问：
1. API 配置（可选）
2. 推荐时间设置（固定时间/随机时间/不设置）
3. 语言风格选择（默认语气/人格文件）

### Agent 兼容性

#### Claude Code
- ✅ 支持交互式 UI 界面（AskUserQuestion 工具）
- ✅ 用户可以直接点击选择选项
- ✅ 更好的用户体验

#### opencode
- ✅ 支持交互式 UI 界面（UserInput/SelectOption 工具）
- ✅ 用户可以直接点击选择选项
- ✅ 更好的用户体验

#### 其他 Agent（hermes、openclaw 等）
- ✅ 支持交互式 UI 界面（如果 agent 提供）
- ✅ 自动回退到文本提示模式（如果不支持）
- ✅ 用户通过回复数字选择选项

**注意**：
- 支持多种 UI 工具：AskUserQuestion、UserInput、SelectOption
- 自动检测 agent 支持的 UI 工具
- 如果不支持 UI，自动回退到文本提示模式
- 两种模式功能完全相同，只是交互方式不同

### 方式2：手动触发
```
/hertz-myself
推荐音乐
今日推荐
```

### 方式3：查看统计
```
/music-stats
查看评分统计
```

### 方式4：使用外部 cron 定时
如果你使用 openclaw、hermes 等机器人，可以使用它们自带的 cron 功能定时调用本技能。

示例 cron 配置：
```bash
# 每天 18:00 推荐
0 18 * * * /path/to/hertz-myself

# 每天 8:00 和 18:00 推荐
0 8,18 * * * /path/to/hertz-myself
```

## 推荐规则

### ✅ 推荐
- 独立音乐
- 小众佳作
- 冷门宝藏
- 实力派歌手
- 创作型歌手
- 歌词有深度
- 旋律有特色
- **全球范围**：中文、英文、日文、韩文、法文、德文等均可
- **不限年代**：经典老歌、新歌均可

### ❌ 过滤
- 抖音热歌
- 网络神曲
- 口水歌
- 洗脑旋律
- 网红歌曲
- 15秒副歌
- 变装BGM
- 挑战BGM
- **避免重复**：最近推荐过的歌曲不再推荐

## 注意事项

1. **实时搜索**：每次推荐都需要 AI 实时搜索，不使用缓存
2. **随机性强**：每次推荐都要不同，避免重复
3. **拒绝热歌**：严格过滤抖音热歌、口水歌、网红歌
4. **全球范围**：不限语言，中文、英文、日文、韩文等均可
5. **不提供链接**：链接容易失效，请在音乐平台上搜索歌名和歌手名称收听
6. **用户反馈**：鼓励用户评分，优化推荐
7. **隐私保护**：评分数据仅存储在本地
